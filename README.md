# Ng2-Socially

The demo following the Meteor - Angular 2 tutorial at [Angular-Meteor.com](http://angular-meteor.com/tutorials/angular2/).

## Get Started

1. Clone the repo and open the folder.

    > git clone git@github.com:ShMcK/NG2-Socially.git
    > cd NG2-Socially
